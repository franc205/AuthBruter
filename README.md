# AuthBruter
This tools allows you to brute force the "auth token" on Twitter.

#Instalation

- git clone https://github.com/franc205/AuthBruter.git
- chmod +x AuthBruter/AuthBruter.py
- AuthBruter

#Requirements

- Python 2.7
- An operating system (tested on Elementary OS)
